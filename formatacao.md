# Formatação
<hr>
Atualmente a formatação em HTML ficou muito resetrita, tendo pouco versatilidade. Agora o responsável pela formatação é o arquivo CSS

## Título pré-definido
`<h1>Título</h1>`<br>
`<h2>Título</h2>`<br>
`<h3>Título</h3>`<br>
`<h4>Título</h4>`<br>
`<h5>Título</h5>`<br>
`<h6>Título</h6>`<br>

Exemplo:

<h1>Título</h1><br>
<h2>Título</h2><br>
<h3>Título</h3><br>
<h4>Título</h4><br>
<h5>Título</h5><br>
<h6>Título</h6><br>
