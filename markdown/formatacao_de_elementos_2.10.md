## Formatação de Elementos

### Posicionamento
A tag `<DIV>` nos auxilia com o posicinamento dos elementos <br>
<b>Propriedades gerais da div</b><br>
Exemplo:

    div{
    border-style: solid;
    border-width: 2px;
    position: absolute;
    }

    .quadro1{
        top: 0px;
        width: 202px;
        height: 102px;
        background-color: yellow;
    }
    .quadro2{
        top: 102px;
        width: 100px;
        height: 100px;
        background-color: red;
    }
    .quadro3{
        top: 102px;
        left: 110px;
        width: 100px;
        height: 100px;
        border-style: dotted;
        background-color: blue;
    }
